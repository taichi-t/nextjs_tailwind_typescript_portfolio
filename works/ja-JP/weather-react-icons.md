## weather-react-icons とは？

OpenWeatherAPI および YahooWeatherAPI で使用する天気アイコンの React コンポーネントを生成する npm ライブラリ。

## 特徴

技術スタックは、React.js、TypeScript、テストには Jest を使用しています。
